# GALAYR ACADEMY - E-Learning Website

A modern e-learning platform built with HTML, CSS, and JavaScript.

## 📚 Overview

**GALAYR ACADEMY** is an online learning platform that provides digital skills education to help students achieve success in their careers.

## 🛠️ Tech Stack

| Category | Technology |
|----------|------------|
| **HTML** | HTML5 |
| **CSS** | CSS3, Bootstrap 5, Owl Carousel |
| **JavaScript** | jQuery, Vanilla JS |
| **Fonts** | Google Fonts (Lato, Poppins) |
| **Icons** | Font Awesome 6.7.2 |
| **Animations** | AOS (Animate On Scroll) |

## ✨ Features

- **Responsive Navigation** — Mobile-friendly navbar with hamburger menu
- **Hero Banner** — Animated banner section with call-to-action buttons
- **Course Catalog** — Display of available courses
- **Admission System** — Online admission form/flow
- **Shopping Cart** — Cart functionality for course purchases
- **FAQ Section** — "Ask Us" section for common questions
- **Smooth Animations** — AOS scroll animations throughout

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```

2. **Open in Browser**
   ```bash
   # Simply open index.html in your browser
   ```

3. **Or use a local server**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

## 🔧 Customization

### Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primarycolor: #2C1434;
    --secondarycolor: #FF8E06;
}
```

### Content
- Main content: `index.html`
- Styles: `css/style.css`
- Scripts: `js/script.js`
